# DOCUMENTAÇÃO

Após fazer o download do projeto via `.zip` ou `git clone`, siga os passos abaixo:

1. **Entre na pasta `Frontend`:**

   cd Frontend
Instale as dependências do projeto:

npm install
Inicie o servidor de desenvolvimento:
npm run dev


Entre na pasta Backend:
cd ../Backend
Instale as dependências do projeto:

npm install
Inicie o servidor:
npm run start
